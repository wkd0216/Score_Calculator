# **App Name**: 성적 메이트

## Core Features:

- Course & Grade Entry: Users can easily add, edit, or remove courses, including course names, credit units, and corresponding letter grades.
- Grade Point Conversion: Automatically converts entered letter grades (e.g., A+, B-) into standard numerical grade points (e.g., 4.5, 3.0).
- Semester GPA Calculation: Calculates the overall Grade Point Average for the entered courses in the current semester.
- GPA & Course Summary Display: Presents the calculated semester GPA prominently, along with a clear list summarizing all entered courses, their credits, and grades.

## Style Guidelines:

- The primary color, #5E7EBA (a soft, professional blue), evokes clarity and academic focus, fitting for a tool aiding students. The HSL value is approximately 220, 40%, 55%.
- The background color, #F0F1F5 (a very light, almost white blue-gray), maintains a clean and unobtrusive canvas, supporting easy readability for academic content. The HSL value is approximately 220, 20%, 95%.
- An accent color, #22A9C3 (a vibrant cyan-blue), is used for highlights and interactive elements, adding a modern and engaging touch. The HSL value is approximately 190, 70%, 45%.
- The font 'Inter' (sans-serif) is recommended for both headlines and body text, providing a modern, objective, and highly legible experience ideal for a data-driven application.
- Simple, clean, and intuitive line icons should be used to represent actions like adding courses, editing, and deleting, maintaining a functional aesthetic.
- A responsive, single-column layout with clear section breaks ensures ease of use on various devices, guiding the user through course entry and result viewing linearly.
- Subtle, quick transition animations on data input and GPA recalculation will provide feedback and enhance the perceived responsiveness of the application.
